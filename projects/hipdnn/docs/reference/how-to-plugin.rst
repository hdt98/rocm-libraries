***********************************
hipDNN guides for plugin developers
***********************************

These guides demonstrate how to use hipDNN's extended functionality, namely developing plugins for the platform.

.. important::

  These instructions are appropriate for plugin developers who have experience with hipDNN, and know how to build and compile a shared library (Linux) or DLL (Windows).
  If you're a new to hipDNN, see :ref:`how-to-users`.
