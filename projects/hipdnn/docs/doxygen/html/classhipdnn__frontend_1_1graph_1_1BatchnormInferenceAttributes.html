Detailed Description

Configuration for batch normalization inference.

BatchnormInferenceAttributes specifies the parameters for batch normalization during inference, using pre-computed running mean and inverse variance:

y = scale * (x - mean) * inv_variance + bias

auto y = graph.batchnorm_inference(x, mean, inv_variance, scale, bias,
BatchnormInferenceAttributes());
