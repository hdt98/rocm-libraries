Detailed Description

Configuration for convolution weight gradient (backward filter) operations.

ConvWgradAttributes specifies the parameters for computing the gradient of the convolution with respect to the weights (wgrad operation): dw = conv_wrw(x, dy)

auto dw = graph.conv_wgrad(dy, x, ConvWgradAttributes()
.set_padding({1, 1})
.set_stride({1, 1}));
