*****************************************
hipDNN installation for plugin developers
*****************************************

This guide demonstrates how to install and configure hipDNN from source on Linux or Windows.

.. important::

  These instructions are appropriate for plugin developers who have experience with hipDNN, and know how to build and compile a shared library (Linux) or DLL (Windows).
  If you're a new to hipDNN, see :ref:`user-install`.
