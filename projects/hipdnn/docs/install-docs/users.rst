*****************************
hipDNN installation for users
*****************************

This guide demonstrates how to install hipDNN from a CMake project.

.. important::

  These instructions are appropriate for general-purpose users who may be new to hipDNN. 
  If you're experienced with hipDNN and want to develop plugins for the platform, see :ref:`plugin-install`.
