Detailed Description

Data type identifiers for hipDNN backend attribute values.

This file defines the types used to specify the data type of attribute values when calling hipdnnBackendSetAttribute() and hipdnnBackendGetAttribute().

Enumeration Type Documentation

◆ hipdnnBackendAttributeType_t

enum hipdnnBackendAttributeType_t

Data types for backend descriptor attribute values.

When setting or getting attributes on backend descriptors, you must specify the data type of the value being passed. These constants identify the expected type for proper marshalling.
